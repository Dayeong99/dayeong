# Linux Backup Shell Script Project
# 운영체제 과제

## 📌 프로젝트 개요
본 프로젝트는 Linux 환경에서 파일을 자동으로 백업하고 오래된 파일을 관리하기 위한 Bash Shell Script입니다.

총 2개의 스크립트로 구성되어 있습니다:
- backup.sh
- safebackup.sh

---

## 📁 파일 설명

### 1️⃣ backup.sh
기본적인 파일 백업 스크립트입니다.

#### 🔹 기능
- 지정 디렉토리의 오래된 파일 검색
- tar.gz 형태로 압축
- 압축 성공 시 원본 파일 삭제

#### 🔹 특징
- 단순 구조
- 기본 백업 기능 구현
- 사용자 확인 없이 자동 처리

---

### 2️⃣ safebackup.sh
안전 기능이 추가된 향상된 백업 스크립트입니다.

#### 🔹 기능
- 백업 대상 파일 목록 출력
- 사용자 확인 (삭제/압축 전 단계)
- tar.gz 압축 수행
- 삭제 여부 2차 확인 기능
- 안전한 삭제 처리

#### 🔹 특징
- 사용자 확인 단계 추가
- 데이터 손실 방지 구조
- 실무형 안전 백업 스크립트

# Linux Security Check Script (U-15 / U-25)
# 시스템보안 과제

## 📌 과제 설명
본 프로젝트는 Linux 시스템의 보안 점검 항목인 U-15, U-25를 Bash Shell Script로 자동 점검하는 과제입니다.

---

## 📁 파일 구성
- U-15.sh : 계정 및 파일 권한 점검 스크립트
- U-25.sh : 로그인 및 접근 로그 점검 스크립트

---

## 🔐 U-15 점검 항목
- /etc/passwd 파일 권한 확인
- /etc/shadow 파일 권한 확인
- root 계정 정보 확인

---

## 🔐 U-25 점검 항목
- 최근 로그인 기록 확인 (last)
- sudo 권한 사용자 확인
- 로그인 실패 기록 확인 (auth.log)
